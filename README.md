

# **Point Of Sale System**

This is a simple web-based open source software written in PHP and MySQL. It is specifically designed for Point of Sale and can be easily installed on your own server.

If you find any bugs, feel free to send a pull request.
If you have any suggestions, feel free to create issues.

---

## **Installation**

The web application is a simple implementation of a Point of Sale System using PHP and MySQL. To run the application:

1. Download and install XAMPP.
2. Copy the whole project folder to `C:\xampp\htdocs`.
3. Open XAMPP and start Apache and MySQL.
4. Click "Admin" for MySQL to open phpMyAdmin.
5. Create a database with the same name as the SQL file in the project folder.
6. Import the `testdb.sql` file into the database.
7. Open a browser and go to `localhost/project`.
8. Enter some sample information to get started.

## **Purpose**

The Point-of-Sale system is designed to keep track of sales and calculate item prices easily. It updates inventory counts when items are sold and provides an organized way to manage store transactions.

---

## **Goals and Objectives**

* Access store sales history and analytics in one place.
* Provide instant pricing and faster checkout.
* Filter reports by price or date.
* Save time for other business operations.
* Track operator sales reports efficiently.

---

## **Project Description**

The system is a web-based Point of Sale application. Users can track sales, calculate prices, update inventory, add products, and generate billing reports. Operators can manage products, while admins have full system control.

---

## **Scope**

The system focuses on Point-of-Sale operations but can also be adapted for other sales management applications.

---

## **System Users**

* **Administrators:** Full access to all system functions.
* **Operators:** Access limited to functions assigned by the admin.

---


## **Technology**

* **Frontend:** Bootstrap, CSS, HTML
* **Backend:** PHP, MySQL
